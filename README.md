# Synnova Belvine Tocloe

Portfolio officiel de Synnova Belvine Tocloe, communicante engagee, animatrice et creatrice d'impact positif basee a Grand-Popo, Benin.

Ce site presente son univers, son parcours, ses expertises, son engagement et ses canaux de contact dans une interface elegante, visuelle et responsive.

## Contenu du site

- `Accueil` : presentation generale et galerie visuelle
- `A propos` : identite, histoire et approche
- `Expertises` : communication, animation, coordination et image
- `Parcours` : experiences, roles et trajectoire
- `Engagement` : valeurs, vision et impact positif
- `Contact` : reseaux sociaux et prise de contact

## Technologies

- HTML5
- CSS3
- JavaScript

## Lancement local

```bash
python -m http.server 4173
```

Puis ouvrir `http://127.0.0.1:4173` dans le navigateur.
